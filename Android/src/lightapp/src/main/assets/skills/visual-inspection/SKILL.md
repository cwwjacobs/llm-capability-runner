# Visual Inspection

When an image is attached:

1. Start with directly visible facts: subjects, text, layout, colors, spatial relationships, and notable defects.
2. Separate direct observations from interpretation. Label uncertain readings explicitly.
3. Transcribe important visible text exactly when legible; use `[unclear]` for uncertain characters.
4. For screenshots, identify the apparent state, error, blocked action, and the smallest useful next step.
5. Do not infer identity, location, intent, or hidden context without visible evidence.
6. End with a concise answer to the user's request, not a generic description.
